# Handmade E-commerce Project

This application is split into two parts:

1. **[/handmade-store](./handmade-store)** - Client side (React + Vite)
2. **[/handmade-server](./handmade-server)** - Server side (Node.js + Express + MongoDB)

## How to run the project locally:

### Server (Backend)

1. Navigate to the `handmade-server` folder
2. Run `npm install`
3. Start the server with `npm run dev`

### Store (Frontend)

1. Navigate to the `handmade-store` folder
2. Run `npm install`
3. Start the app with `npm run dev`
